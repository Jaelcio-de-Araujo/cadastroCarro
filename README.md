# Sistema de cadastro de carros feito em C

O sistema pode cadastrar até 10 carros(isso pode ser alterado), realizar a buscar por modelo, placa e ano de fabricação.

Usamos alguns recursos da linguagem C :

### Recursos:

* `for`
* `if` 
* `struct `
* `switch `
* etc...

### Bibliotecas:

* `#include <stdio.h>`

* `#include <stdlib.h>`

* `#include <locale.h>`

* `#include <string.h>`

* `#define TAM 10`
